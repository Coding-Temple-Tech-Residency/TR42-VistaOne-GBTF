# Updated Data Collection Requirements Document

This document updates and supersedes the previous data collection requirements. See also event_tracking_plan.md and kpi_list.md.

## Data to Collect

- User lifecycle events (register, login, logout)
- Job lifecycle events (create, assign, complete)
- Submission and photo events
- API usage and error events
- Audit logs for all sensitive actions

## Privacy & Compliance

- No PII or sensitive data in logs
- All data collection must comply with GDPR/CCPA
- Data minimization and retention policies must be documented
