# KPI List with Definitions

| KPI Name                | Definition                                      |
|-------------------------|-------------------------------------------------|
| User Registrations      | Number of new users registered per period       |
| Active Users            | Users who logged in during a period             |
| Jobs Created            | Number of jobs created                          |
| Jobs Completed          | Number of jobs marked as completed              |
| Submission Rate         | Submissions per job                             |
| Photo Upload Rate       | Photos uploaded per submission                  |
| API Error Rate          | Errors per 1000 API calls                       |
