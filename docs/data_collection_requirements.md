# Data Collection Requirements Document

## Data to Collect

- User registrations, logins, logouts
- Job creation, assignment, completion
- Submissions and photos
- Audit logs (actions, timestamps)
- API usage and error events

## Privacy & Compliance

- No PII or sensitive data in logs
- All data collection must comply with GDPR/CCPA
